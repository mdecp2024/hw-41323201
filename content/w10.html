題目二

Python 變數命名的規則:

  1. 以字母或下劃線開頭：變數名稱必須以字母（a-z 或 A-Z）或下劃線（_）開頭，後面可以跟數字或字母。
  2. 不能使用 Python 關鍵字：Python 的關鍵字（如 if、for、while 等）有特殊的語法意圖，不能用作變數名稱。
  3. 不區分大小寫：Python 是大小寫敏感的，因此 my_var 和 MY_VAR 是兩個不同的變數。
  4. 避免使用特殊符號：變數名稱中只能使用字母、數字和下劃線，不能包含如 @, #, !, * 等特殊字符。
  5. 遵循 PEP 8 風格指南：根據 Python 官方的編碼風格指南，變數名稱應使用小寫字母和下劃線分隔單詞（例如 user_name），類別名稱使用駝峰式命名（例如 MyClass）。

針對 Python 程式命名的規則編寫程式:

import keyword

# 1. 顯示所有 Python 的關鍵字
print("Python 關鍵字 (保留字):")
print(keyword.kwlist)  # 輸出所有關鍵字

# 2. 合法的變數名稱（遵守命名規範）
user_name = "Alice"
user_age = 30
PI = 3.14159
print(f"使用者名稱: {user_name}, 年齡: {user_age}, 圓周率: {PI}")

# 3. 使用非法的變數名稱（關鍵字）
# 關鍵字 'if' 是 Python 的保留字，這裡會引發 SyntaxError
# if = "非法變數名稱"  # 取消註解這一行會引發錯誤

# 4. 使用單下劃線開頭的變數（表示內部變數）
_private_var = "這是內部變數"
print(f"內部變數: {_private_var}")

# 5. 使用雙下劃線開頭的變數（名稱重整）
class MyClass:
    def __init__(self):
        self.__private_attribute = "隱藏屬性"
    
    def get_private_attribute(self):
        return self.__private_attribute

my_object = MyClass()
print(f"透過方法訪問隱藏屬性: {my_object.get_private_attribute()}")

# 6. 使用單下劃線作為不關心的變數
for _ in range(3):  # 我們只關心迴圈次數，不關心具體的值
    print("這是一次迴圈執行")
