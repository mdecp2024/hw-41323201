w11_hw

第一題的程式

1. 匯入模組

import math
math 是 Python 的數學模組，用來執行數學相關的計算。

在這段程式中，我們用到了 math.sqrt 函數來計算平方根，這是用來測量點與圓心之間的距離。


2. 定義變數

size = 10
radius = size // 2
center = radius - 0.5
size 是顯示區域的大小，設定為 10，表示會有 10 行與 10 列。

radius 是圓的半徑，設定為 size // 2（整數除法），所以半徑為 5。

center 是圓心的座標，計算為 radius - 0.5，這樣可以更接近圓的中心位置，避免偏差。


3. 雙層迴圈：生成圖案

size = 10
radius = size // 2
center = radius - 0.5
外層迴圈控制 列 (y)，內層迴圈控制 行 (x)，遍歷區域的每個點。

range(size) 會產生從 0 到 size-1 的整數範圍。


4. 計算與圓心的距離

distance = math.sqrt((x - center) ** 2 + (y - center) ** 2)
每個點 (x, y) 到圓心 (center, center) 的距離使用公式計算： 距離=(x−圓心x)2+(y−圓心y)2

將這個點的座標代入後，距離會決定該點是否屬於圓內。


5. 判斷並繪製圖案

if distance <= radius:
    print(" * ", end="")
else:
    print("   ", end="")
如果距離 distance 小於或等於圓的半徑 radius，表示該點在圓內，則打印 " * "。

否則打印空白 " "，用來填充圓外的區域。

end="" 確保在同一行內打印多個字符。


6. 換行

print()
內層迴圈完成一行後，透過這行程式碼換行，準備打印下一行。

第二題的程式

1.導入所需的模組

from browser import html 
from browser import document as doc

browser.html：用於建立和操作 HTML 元素。

browser.document：用於操作 HTML DOM 文件（例如存取或修改網頁上的元素）。

2.建立一個畫布 (canvas) 元素

canvas = html.CANVAS(width=400, height=400)
brython_div = doc["brython_div1"]
brython_div <= canvas

建立一個 400×400 像素大小的 <canvas> 元素。

找到 HTML 中的 brython_div1 元素，並將畫布加入這個元素中。

3.定義全域參數

gs = 20  # 每一格的像素大小
center = (5, 5)  # 圓的中心點座標（格子單位）
radius = 5  # 圓的半徑（格子單位）

gs：每個格子邊長為 20 像素，畫布總共是 10×10 格（400 ÷ 20）。

center：圓心位置以格子為單位，(5, 5) 是中心。

radius：圓的半徑是 5 格。

4.獲取畫布的 2D 繪圖上下文

ctx = canvas.getContext("2d")

ctx 是畫布的繪圖上下文，提供了用來繪製圖形的方法。

5.畫矩形框線

def dRect(lux, luy, w, h, s=1, c='#ff0000'):
    ctx.lineWidth = s
    ctx.strokeStyle = c
    ctx.beginPath()
    ctx.rect(lux, luy, w, h)
    ctx.stroke()
dRect 是一個用來繪製矩形框線的函數：
  1. (lux, luy)：矩形的左上角位置。
  2. w, h：矩形的寬和高。
  3. s：框線的寬度。
  4. c：框線的顏色（預設是紅色）。

使用 ctx.rect() 定義矩形，然後用 ctx.stroke() 繪製邊框。

6. 繪製網格

def grid(width, height, grid_pix):
    for i in range(width):
        for j in range(height):
            dRect(i * grid_pix, j * grid_pix, grid_pix, grid_pix, 1, "lightgrey")
繪製整個畫布的網格：
  1. 以 width × height 的格子數量進行迴圈。
  2. 每個格子大小為 grid_pix × grid_pix（此例中是 20 像素）。
  3. 每個格子框線使用淺灰色。


7. 填充格子

def fill(x, y, color):
    ctx.fillStyle = color
    ctx.fillRect(x, y, gs, gs)
fill 函數用於填充一個格子：
  1. (x, y) 是格子的左上角座標。
  2. color 指定填充的顏色。
  3. 使用 ctx.fillRect() 填滿。


8. 繪製圓形

def drawCircle(center, radius, gs):
    for y in range(10):
        for x in range(10):
            cx = x + 0.5  # 格子的中心 x 座標
            cy = y + 0.5  # 格子的中心 y 座標
            dist = ((cx - center[0]) ** 2 + (cy - center[1]) ** 2) ** 0.5
            if dist <= radius:
                fill(x * gs, y * gs, "black")
用來繪製圓形，實現方法：
  1. 使用兩層迴圈遍歷 10×10 的格子。
  2. 計算每個格子中心 (cx, cy) 與圓心 (center[0], center[1]) 的距離。
  3. 如果距離小於等於半徑，則填充該格子為黑色。


9. 繪製網格與圓形

grid(10, 10, gs)
drawCircle(center, radius, gs)
先繪製整個網格，然後在網格上繪製圓形。
