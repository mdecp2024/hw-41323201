Python 變數命名的規則:




    

  1. 以字母或下劃線開頭：變數名稱必須以字母（a-z 或 A-Z）或下劃線（_）開頭，後面可以跟數字或字母。
    2. 

  2. 不能使用 Python 關鍵字：Python 的關鍵字（如 if、for、while 等）有特殊的語法意圖，不能用作變數名稱。
    3. 

  3. 不區分大小寫：Python 是大小寫敏感的，因此 my_var 和 MY_VAR 是兩個不同的變數。
    4. 

  4. 避免使用特殊符號：變數名稱中只能使用字母、數字和下劃線，不能包含如 @, #, !, * 等特殊字符。
    5. 

  5. 遵循 PEP 8 風格指南：根據 Python 官方的編碼風格指南，變數名稱應使用小寫字母和下劃線分隔單詞（例如 user_name），類別名稱使用駝峰式命名（例如 MyClass）。

第一題的程式


1. 匯入模組


import math




math 是 Python 的數學模組，用來執行數學相關的計算。




在這段程式中，我們用到了 math.sqrt 函數來計算平方根，這是用來測量點與圓心之間的距離。



2. 定義變數


size = 10
radius = size // 2
center = radius - 0.5




size 是顯示區域的大小，設定為 10，表示會有 10 行與 10 列。




radius 是圓的半徑，設定為 size // 2（整數除法），所以半徑為 5。


center 是圓心的座標，計算為 radius - 0.5，這樣可以更接近圓的中心位置，避免偏差。



3. 雙層迴圈：生成圖案






size = 10
radius = size // 2
center = radius - 0.5

外層迴圈控制 列 (y)，內層迴圈控制 行 (x)，遍歷區域的每個點。




range(size) 會產生從 0 到 size-1 的整數範圍。





4. 計算與圓心的距離


distance = math.sqrt((x - center) ** 2 + (y - center) ** 2)




每個點 (x, y) 到圓心 (center, center) 的距離使用公式計算： 距離=(x−圓心x)2+(y−圓心y)2




將這個點的座標代入後，距離會決定該點是否屬於圓內。





5. 判斷並繪製圖案


if distance <= radius:
    print(" * ", end="")
else:
    print("   ", end="")





如果距離 distance 小於或等於圓的半徑 radius，表示該點在圓內，則打印 " * "。




否則打印空白 " "，用來填充圓外的區域。


end="" 確保在同一行內打印多個字符。





6. 換行


print()




內層迴圈完成一行後，透過這行程式碼換行，準備打印下一行。

8. 繪製圓形


def drawCircle(center, radius, gs):
    for y in range(10):
        for x in range(10):
            cx = x + 0.5  # 格子的中心 x 座標
            cy = y + 0.5  # 格子的中心 y 座標
            dist = ((cx - center[0]) ** 2 + (cy - center[1]) ** 2) ** 0.5
            if dist <= radius:
                fill(x * gs, y * gs, "black")




用來繪製圓形，實現方法：




    

  1. 使用兩層迴圈遍歷 10×10 的格子。
    2. 

  2. 計算每個格子中心 (cx, cy) 與圓心 (center[0], center[1]) 的距離。
    3. 

  3. 如果距離小於等於半徑，則填充該格子為黑色。
    4.
